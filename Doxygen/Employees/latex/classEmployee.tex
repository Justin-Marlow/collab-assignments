\doxysubsection{Detailed Description}
Represents an employee with basic information and functionality. 

The \mbox{\hyperlink{classEmployee}{Employee}} class stores an employee\textquotesingle{}s ID, years of service, hourly rate, and hours worked. It provides functions to print employee info, calculate pay, and simulate an anniversary raise.
